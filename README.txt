- 실행법 -
병렬화 코드는 parallel 디렉토리에 저장되어 있으며,
make 명령어를 통해 실행 파일(main_3) 를 생성하고
해당 디렉토리에서 './main_3' 를 입력해 실행할 수 있다.

순차 코드는 sequential 디렉토리에 저장되어 있으며,
make 명령어를 통해 실행 파일(main_3) 을 생성하고
해당 디렉토리에서 './main_3' 을 입력해 실행할 수 있다.

각각 입력 파일은 'main_v*.c' 에 DATA 라는 이름으로 정의되어 있으며,
해당 부분을 변경함으로써 입력 파일 변경이 가능하다.
테스트용 파일은 https://1drv.ms/f/s!AqXVgZVfh1CAwSHnZu7aBDTtIeBK 에서 확인이 가능하다.
